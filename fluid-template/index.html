About this site

My calling card is this site. It's fast and responsive and uses no external libraries that would slow things down. My philosphy is to work with the (now amazing) capabilities of the browser and write only the javascript required. I'm tired of bloat, especially when it comes to mobile.

If you're a developer you might find a version of this code a good starting point for your next HTML5 project. The source, along with a few example layouts is available on Github. I've also used this to make the Jeckyll theme that powers my blog.
